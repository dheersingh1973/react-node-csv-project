# Backend

This folder contains the backend services for the application.

## Technologies Used
- Node.js
- Express.js
- MongoDB (via Mongoose)

## Setup
1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
